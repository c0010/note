** api doc
   文件信誉与域名信誉 api v2 文档说明
   
   - 文件信誉

      /api/v2/file?apikey=manue1sec&param=3b405c30a7028e05742d0fbf0961e6b2&timestamp=1525682461
     
     #+BEGIN_CENTER
       | 参数      | 参数说明                                    |
       |-----------+---------------------------------------------|
       | apikey    | token认证信息                               |
       |-----------+---------------------------------------------|
       | param     | 样本MD5、sha1                               |
       |-----------+---------------------------------------------|
       | timestamp | 用户请求当前时间戳,服务的处理最近五分钟请求 |
       |-----------+---------------------------------------------|
     #+END_CENTER

       返回状态码同api v1
