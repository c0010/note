** config Manage key
SPC f e d 快速打开.spacemacs

** file Manage key
SPC f f 打开文件夹搜索 C-h 
SPC f r open recent file 
SPC f R rename file 
SPC f c copy file current file 
SPC f j open dired 
SPC p f 在当前文件夹下搜索文件名 
SPC f t 打开/关闭侧边栏
** Org-mode GTD key

C-c C-d   添加结束时间
C-c .     添加时间段
C-a       GTD 管理列表
